Register Product BPOM And Halal

FLOW MENU BPOM
DRAFT
  • Mengisi form bpom berdasarkan brand product
  • Pada Status DRAFT PRODEV Melakukan approve ke SUPER RA
SUPER RA
  • Mengevaluasi data approve dari prodev
  • Selanjutnya Admin Super RA Melakukan Approve ke ADMIN RND
ADMIN RND
  • Mengevaluasi data approve dari prodev
  • Selanjutnya Admin Super RA Melakukan Approve ke ADMIN RND
ADMIN RA
  • Mengevaluasi data approve dari ADMIN RND
  • Melakukan Reject Jika Data Dievaluasi Tidak Sesuai Standart
REJECT
  • Di Status Reject Prodeve Melakukan Perbaikan Data
  • Prodev Melakukan Resubmit Ke SUPER RA
SUBMITTED
  • Admin RA Mengupload Data Certificate
  • Selanjutnya Di Approve ke Submited
NOTIFIED
  • Data Sudah Notified
  • Data Di Notified Sudah Final
FLOW MENU HALAL
  • Sertifikat Hanya Diterbitkan oleh Admin Halal
REPORT
  • Report pendaftaran bpom bisa ditarik berdasarkan kategori filter
